# Changelog

All notable changes to the DeepLens AI project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-25

### Added
- **AI Core (CViT & CViT2)**: Integration of Convolutional Vision Transformer deepfake detection models supporting single-frame and sequence inference.
- **FastAPI Backend**: Built-in endpoints for file uploads, video frame splitting, BlazeFace face detection, and real-time deepfake classification.
- **React Frontend**: Premium interactive user interface featuring real-time detection logs, an interactive analytics dashboard, a local history manager, and comprehensive PDF report generation.
- **Academic Integration**: Direct reference links and automated BibTeX copying for the CViT (ArXiv '21) and CViT2 (IEEE GEM '24) papers.
- **Repository Setup**: Initial configuration with MIT license, professional documentation (`README.md`, `CONTRIBUTING.md`, `PROJECT_STRUCTURE.md`), and optimized structure.
